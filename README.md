# Jest-Workshop
An interactive workshop about the many uses of Jest

Preperation:
```bash
$ export NODE_TLS_REJECT_UNAUTHORIZED=0 && npm ci
```

Snapshot API:
```bash
$ npm run test:api -- --watchAll
```

Snapshot APP:
```bash
$ npm run test:api -- --watchAll
```

Cleanup:
```bash
$ unset NODE_TLS_REJECT_UNAUTHORIZED
```
